# Pas compatible

_**Est-ce que c'est compatible avec le plugin X?**_

Je ne peux pas répondre sûrement à cette question parce que je ne peux pas savoir comment chaque plugin dans le monde est codé, mais voici une liste de plugins qui peuvent causer des problèmes:

* Tous les plugins qui utilisent des **resourcepacks personnalisés** (vous pouvez les rendre compatibles si vous avez des connaissances minimales sur la façon de fusionner les resourcepacks manuellement, assurez-vous de ne pas remplacer les fichiers ItemsAdder et c'est bon)
* [CraftEnhance](https://www.spigotmc.org/resources/custom-recipes-and-crafting-craftenhance.65058/), ce plugin perturbe la logique des recettes personnalisées d'ItemsAdder et crée des bugs de duplication. Donc, s'il vous plaît, ne l'utilisez pas
* Les plugins qui personnalisent le comportement et les recettes de la table de craft
* Pour l'instant ce n'est **pas compatible** avec les **plugins** et les générateurs de mondes qui **génèrent des blocs de champignons** avec des faces différentes pour créer des textures personnalisées. À l'avenir, j'ajouterai la compatibilité.
